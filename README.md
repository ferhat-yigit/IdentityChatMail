# 📬 Identity Mail & Chat Projesi
📚Bu proje, .NET Core 8.0 ile geliştirilmiş Kapsamlı İletişim Yönetim Platformudur. Kullanıcılar, sisteme kayıt olup giriş yaptıktan sonra e-posta gönderme, alma, silme ve okundu olarak işaretleme işlemleri gerçekleştirebilmektedir. Proje, Entity Framework Core ve Identity altyapıları ile desteklenmektedir.

Identity Email & Chat Projesi, M&Y Akademi'deki eğitim sürecimde, eğitmen Murat Yücedağ'ın rehberliğinde geliştirildi.

🗂️**Kullanılan Teknolojiler**
-  ASP.NET Core 8.0
- Entity Framework Core (Code First)
- ASP.NET Core Identity – Kullanıcı yönetimi ve güvenlik sistemleri
- Razor Views & Partial Views – Dinamik ve sürdürülebilir kullanıcı arayüzleri
- LINQ & Contains() – Veritabanı sorguları ve hızlı arama filtrelemeleri
- Dependency Injection – İhtiyaç duyulan servislerin esnek şekilde entegrasyonu
- Authentication & Authorization – Güvenli giriş ve kullanıcı yetkilendirme yönetimi
- E-posta Doğrulama & Parola Sıfırlama – Kullanıcı doğrulama ve şifre yenileme
-  Şifreleme ve Güvenlik – Parolaların güvenli saklanması ve parola hash depolama
- Kullanıcı Kayıt ve Giriş Sayfası
